# Vibe-Code-Practice

## ✨ Vibe Checklist App

A simple, lovable, and complete checklist application built with vanilla HTML, CSS, and JavaScript.

### Features

- ✅ Add and remove tasks
- ✅ Mark tasks as complete/incomplete
- ✅ Persistent storage (localStorage)
- ✅ Clean, modern UI with beautiful gradients
- ✅ Responsive design for mobile and desktop
- ✅ Keyboard shortcuts (Ctrl/Cmd + Enter to add, Escape to clear)
- ✅ Task statistics
- ✅ Smooth animations and transitions

### Getting Started

1. Open `index.html` in your web browser
2. Start adding tasks to your checklist!
3. Click the checkbox to mark tasks as complete
4. Click the trash icon to delete tasks

### Keyboard Shortcuts

- **Ctrl/Cmd + Enter**: Add task
- **Escape**: Clear input field

### Tech Stack

- HTML5
- CSS3 (with modern features like CSS Grid, Flexbox, and CSS Variables)
- Vanilla JavaScript (ES6+)
- Local Storage for data persistence

### Development

This is a vanilla web application that doesn't require any build tools or dependencies. Simply open `index.html` in a browser to run it locally.

The code is organized into three main files:
- `index.html` - Structure and markup
- `styles.css` - Styling and animations
- `script.js` - Functionality and interactions
